# アレンジしてオリジナル問題を作成
大富豪の A 君は、独特な収集癖を持っています。
今回はどういうわけか、1個 a 円のまったく同じデザインの鞄を、なんと b 個も爆買いしてしまいました。
「同じ鞄をそんなにどうするの？」という周囲のツッコミはさておき、A 君がこの奇妙な爆買いに合計で何円費やしたかを計算するプログラムを書いて、
彼の財布の心配をしてあげてください。

import sys


def main():
    # 標準入力から1行ずつ読み込む
    # 1行目: 鞄1個の値段 (a)
    a = int(sys.stdin.readline().strip())

    # 2行目: 買った鞄の個数 (b)
    b = int(sys.stdin.readline().strip())

    # 合計金額を計算 (a円 × b個)
    total_cost = a * b

    # 結果を出力
    print(total_cost)


if __name__ == "__main__":
    main()
