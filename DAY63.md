# アレンジしてオリジナル問題を作成
あなたはバトル漫画の主人公です。ついに新しい必殺技を思いつきましたが、技名が普通すぎていまいち威厳が足りません。
そこで、技名の末尾に「魂の1文字」を加えることで、より強そうな技名にパワーアップさせることにしました。
元々の技名を表す文字列 S と、末尾に追加する1文字 cが改行区切りで与えられます。完成した最強の必殺技名を出力するプログラムを作成してください。

## 入力例
NekoPunch
Z

## 出力例
NekoPunchZ

### 解答コード
def main():
    # 元々の技名 S を取得
    S = input()
    
    # 末尾に追加する1文字 c を取得
    c = input()
    
    # 文字列の結合と出力
    print(S + c)

if __name__ == "__main__":
    main()
