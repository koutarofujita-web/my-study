# Linuxサーバー初期構築とセキュリティ防衛・監視の実践

Linux（Ubuntu/Debian系）環境において、一般ユーザーの権限隔離から、Webサーバー（Nginx）の構築、
ファイアウォール（UFW）によるアクセス制御、そしてログのリアルタイム解析まで、インフラ・セキュリティ運用の現場（SOCなど）で
必須となる一連のサーバー初期構築プロセスを実践・検証しました。



## 🛠️ 本プロジェクトで実装・検証したアーキテクチャ

1. **最小特権の原則に基づくユーザー隔離** (`useradd`, `passwd`, `su`)
2. **Web公開基盤の構築** (Nginxによる80番ポートの開放)
3. **パケットフィルタリングによる防御陣形の形成** (UFWによるホワイトリスト化)
4. **リアルタイム・セキュリティ監視** (`tail -f` によるアクセスログ監査)

---

## 📝 各フェーズの実行コマンドと技術検証

### Phase 1: ユーザー管理と権限隔離（セキュリティの要）
システムを誤操作や不正アクセスから守るため、管理者（root）権限を持たない一般作業用ユーザーを作成し、Linuxの堅牢な権限隔離機構を検証しました。
