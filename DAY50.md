# アレンジをしてオリジナル問題を作成

## 問題
ここは、果物計算の厳密さで知られる「アマゾン大密林」。
大金持ちのナマケモノであるあなたは、大好物のバナナを「箱買い」することにしました。

この世界では、バナナは鮮度を保つために **1箱にちょうど 12本（＝1ダース）** ずつ綺麗に梱包されて届きます。

しかし、あなたはナマケモノ。あまりの眠気に、自分が「合計で何箱（何ダース）のバナナを買ったか」は覚えているものの、
**「結局、全部で何本のバナナがあるのか」** を計算する前に寝落ちしてしまいました。

あなたの代わりに、購入した総ダース数から、バナナの「総本数」を計算するプログラムを書いて助けてあげてください！

---

## 📥 入力される値
入力は以下のフォーマットで標準入力から与えられます。

```text
N
import sys

def main():
    # 標準入力から総ダース数（N）を取得
    line = sys.stdin.read().strip()
    if not line:
        return
        
    n = int(line)
    
    # 1ダース=12本を掛け算して総本数を算出
    total_bananas = n * 12
    
    # 結果を出力
    print(total_bananas)

if __name__ == '__main__':
    main()
