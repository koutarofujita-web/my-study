## アレンジしてオリジナル問題を作成。

### 問題文
あなたは、レストラン「フレンチ・ビストロ」にやってきました。
お腹がペコペコのあなたのお目当ては、 **「hamburg（ハンバーグ）」** です。
しかし、目の前にあるメニューを開いてあなたは絶望しました。料理名が複雑すぎて、どれがハンバーグなのか瞬時に理解できません。
このままでは、お目当ての料理にたどり着く前に空腹になってしまいます。
与えられたメニュー名の中に、あなたの食べたい料理が
**「独立した単語（スペース区切りの完全一致）」**として含まれているかを判定し、
含まれていれば `Yes`、含まれていれていなければ `No` を出力する、
命を救うための高速判定プログラム（パーサー）を作成してください。



### 解答コード
```python3
import sys

def main():
    # 1行目: 食べたい料理
    w = sys.stdin.readline().strip()
    # 2行目: メニュー名
    s = sys.stdin.readline().strip()
    
    # メニュー名をスペースで分割してリスト化
    menu_words = s.split()
    
    # リストの中に食べたい料理が単語として存在するか判定
    if w in menu_words:
        print("Yes")
    else:
        print("No")

if __name__ == '__main__':
    main()
