# アレンジしてオリジナル問題を作成
あなたは一人暮らしの節約家ですが、近所のスーパーで「500円以上買うと、もやし（1包30円）が1つ無料でついてくる」
というバグのようなキャンペーンが始まりました。しかし、あなたは極度の「もやし好き」なため、無料でもらえる権利があるなら
さらにもう1包（30円）を追加で自腹購入するという謎のこだわりを持っています。会計金額 $N$ 円が与えられるので、最終的な支払い金額を計算してください。

## 入力例1
750
## 出力例１
780

### 解答コード
import sys

def main():
    # 入力を取得
    price = int(sys.stdin.read().strip())
    
    # 500円以上なら30円追加
    total = price + 30 if price >= 500 else price
    
    print(total)

if __name__ == "__main__":
    main()
