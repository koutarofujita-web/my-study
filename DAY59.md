あなたは、地球侵略をもくろむ宇宙の帝王「デス・コズミック」のシステム開発部に所属する新米エンジニアです。
ある日、帝王から「地球破壊兵器の起動画面に、カッコいいカウントダウンの日付（月/日）を表示させろ。
ただし、数字だけを入力したら自動でスラッシュを挟んで結合する、画期的な未来システムにするのだ！」と理不尽な仕様書が回ってきました。
もしプログラムがバグって、スラッシュの代わりに計算（割り算）をしてしまったら、兵器の出力が12÷31≒0.387％に大暴落し、
帝王の怒りを買って宇宙のチリにされてしまいます。地球を安全（？）に破壊するため、入力された2つの数字を / で連結して、デス・ディスプレイに美しく出力するプログラムを作成してください

## 入力される値
a
b

### 回答code
# 地球破壊兵器・カウントダウン日付結合モジュール
# 仕様: 帝王の怒りに触れないよう、割り算はせず、文字列として「/」で結合する。

def main():
    # 帝王からの入力を受け取る
    month = input().strip()
    day = input().strip()
    
    # スラッシュで区切ってディスプレイに表示
    print(f"{month}/{day}")

if __name__ == "__main__":
    main()
