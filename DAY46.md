# アレンジ問題を作成

【問題文】
エンジニアのあなたは、先進的（という名の監視社会）なスマートシティ「A 市」に住んでいます。
この街での移動手段は、完全自動運転（たまに暴走する）の「zバス」のみ。
そして、この街の通貨と同期する唯一の決済手段、それがICカード「bカート」です。

毎日、満員バスに揺られてギークハウスとオフィスを往復するあなた。
少しでも交通費を浮かせて、高級キーボード（HHKB）やエナジードリンク（Monster）の資金に
回すため、paicaの「bカート」を限界までハックするアルゴリズムを開発することになりました。

------------------------------------------------------------------------
入力仕様
* 最初の行: 初期チャージ残高 (X 円) と 乗車回数 (N 回)
* 続く N 行: 各乗車で発生する 運賃 (F_i 円)

------------------------------------------------------------------------
 出力仕様
------------------------------------------------------------------------
すべての乗車が終わった後、あなたの paica に残されている
「カード残高（円）」と「paica ポイント（pt）」をスペース区切りで出力してください。

========================================================================
"""

import sys

def main():
    # 標準入力から初期残高と乗車回数を取得 (例: 2000 3)
    input_line = sys.stdin.readline().split()
    if not input_line:
        return
    
    balance = int(input_line[0])
    n = int(input_line[1])
    
    points = 0
    
    # 乗車回数分の運賃を処理
    for _ in range(n):
        fare_str = sys.stdin.readline().strip()
        if not fare_str:
            break
        fare = int(fare_str)
        
        # ポイントが運賃以上ある場合は、ポイントを最優先で使用
        if points >= fare:
            points -= fare
        else:
            # ポイントが足りない場合はカード残高で支払い、10%還元
            balance -= fare
            points += fare // 10  # 小数点以下切り捨て

    # 最終的な残高とポイントを出力
    print(f"{balance} {points}")

if __name__ == '__main__':
    main()
