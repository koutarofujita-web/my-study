# 🖥️ System Health Check CLI Tool

WSL (Ubuntu / Linux) 環境で動作する、軽量かつ視覚的にシステムの状態を確認できる Bash シェルスクリプトです。  
CPU、メモリ、ディスク使用率、ネットワークの応答速度（Ping）、セキュリティ状態を自動的に計測し、星評価と総合評価（A〜D）でリアルタイムに表示します。



---

## ✨ 特徴 (Features)

- **リアルタイム計測**: システムの最新ステータスを即座に取得して評価
- **視覚的なUI**: ANSIカラーコードと星評価（`★★★★★`）で直感的にシステム状態を把握
- **総合評価システム**: 各項目のスコアを合計し、システム全体の健全性を `A`〜`D` で判定
- **依存ライブラリなし**: 標準的な Linux コマンド（`top`, `free`, `df`, `ping` 等）のみで動作

---

## 📊 チェック項目 (Monitored Items)

1. **CPU 使用率** (`top`)
2. **Memory 使用率** (`free`)
3. **Disk 使用率** (`df`)
4. **Network 応答時間** (`ping`)
5. **Security 状態** (`ufw` ステータスチェック)
6. **総合評価 (Rank A - D)**
