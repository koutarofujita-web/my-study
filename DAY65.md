# 🚀 宇宙最果ての配送センター：住所フォーマット変換プログラム

## 📖 ストーリー
西暦3000年、地球外生命体への配達業務を行っている「銀河急便」の新人オペレーターであるあなたは、怪しいエイリアンから住所データの入力を受け取りました。

エイリアンはなぜか「セクター番号」「モジュール番号」「ポッド番号」という3つの数字を、1行ずつ改行して唱えてきます。

```text
（エイリアンの発言）
10
813
100

# 3行の入力を文字列として受け取る（数値計算はしないため int 変換は不要）
sector = input()
module = input()
pod = input()

# f-string を使ってハイフン区切りで結合して出力
print(f"{sector}-{module}-{pod}")

# 別解：print の sep オプションを使う場合
