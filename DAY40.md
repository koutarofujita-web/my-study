## オリジナル問題を作成。

### 問題文

満開の桜を「O(1)」で数えるライフハック（物理）📖 ストーリーある日、私は一本の桜の枝を見つめていました。
そこには、今にも咲きそうな、ぷっくりとしたつぼみが $n$ 個。風情があって大変よろしい。……が、エンジニアの脳内は、
無慈悲にもこう弾き出したのです。「桜の花びらは、1つのつぼみから5枚開く。……じゃあ、全部咲いたら何枚になるんだ？」風情を
完全に数式（$5 \times n$）で殴る、狂気のプログラミングが今、幕を開ける――。


##  Input Specification (入力仕様)






```python
import sys
def count_sakura_petals():
    # 目の前にあるつぼみの数を、血眼になって数えて入力します
    try:
        n = int(input("つぼみの数を入力してね（例: 3）: "))
    except ValueError:
        print("🌸「数字を入力してくれないと、桜も咲くに咲けません」")
        return

    # 1つのつぼみ = 5枚の花びら という大自然の摂理
    total_petals = n * 5

    # 結果を、情緒を添えて出力
    print(f"🌸 すべてのつぼみが開いた時、世界は {total_petals} 枚の花びらで満たされる。")

if __name__ == "__main__":
    count_sakura_petals()
