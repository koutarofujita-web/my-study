 ## オリジナル問題を作成。

 ### 問題
とあるIT企業に勤めるCさんは、今日も定時ギリギリまでオフィスで動画を見て過ごしていました。
しかし、時計を見るとすでに終電の時間が迫っています！Cさんが乗るべき終電は、会社のビルをぐるぐると取り囲む、
謎の「円形超巨大駐車場（1周の距離：M メートル）」の向こう側にある駅。Cさんは必死にその駐車場の周囲を 
$N$ 周 走って、なんとか駅に滑り込みました。
翌朝、筋肉痛で動けなくなったCさんは、自分が一体何メートル走ったのか気になって夜も眠れません。
Cさんの代わりに、走った総距離（メートル）を計算するプログラムを作ってあげてください。

### 入力される値
M （駐車場の1周の距離：メートル）
N（cさんが走った周数）

### 出力code

python3
import sys


def main():
  
    try:
        m = int(input().strip())  
        n = int(input().strip()) 
    except ValueError:
        print
        return

    # 総距離を計算
    total_distance = m * n

    # 結果を出力
    print(total_distance)


if __name__ == "__main__":
    main()
